DESCRIPTION 
-----------
This is a simple script to access pnr-status from the terminal.

USING THE TOOL 
--------------
Type the following commands on a terminal -
<br>`git@github.com:athityakumar/pnr-status.git`
<br> `cd pnr-status` 
<br> `ruby pnr-status.rb`
